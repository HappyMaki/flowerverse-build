# flowerverse

## Setup Instructions
Install git (https://www.atlassian.com/git/tutorials/install-git)

Clone this repository
```
git clone 
cd flowerverse
```

Install python3 (https://docs.python-guide.org/starting/install3/linux/)

Host Server
```
python -m http.server --bind 0.0.0.0 --port 8080
```
